# Day10 FreeRTOS任务拆分：sensorTask 与 uartTask

## 一、实验目标
在 FreeRTOS 工程中完成基础任务拆分，将“数据采集”和“串口输出”分成两个独立任务。

## 二、实验内容
今天完成了以下内容：

- 创建 `sensorTask`
- 创建 `uartTask`
- 使用结构体统一保存系统数据
- `sensorTask` 周期更新数据
- `uartTask` 周期读取并打印数据

## 三、任务设计
### 1. sensorTask
- 功能：周期采样并更新共享数据
- 周期：500ms

### 2. uartTask
- 功能：周期打印当前数据
- 周期：1000ms

## 四、知识点总结
通过今天的实验，主要学习了以下内容：

- 任务职责拆分思想
- 共享数据结构体的基本使用
- 采集任务与输出任务的区别
- 多任务工程中不同任务的协作方式

## 五、实验现象
程序运行后，串口可以周期性输出当前采样值、换算电压和采样次数，说明 `sensorTask` 和 `uartTask` 已经独立运行并完成了基本协作。

## 六、实验结论
今天完成了从“简单任务运行”到“按职责拆分任务”的过渡，为后续学习队列和任务通信打下了基础。
