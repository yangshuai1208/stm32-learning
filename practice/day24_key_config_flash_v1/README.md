# Day24 按键修改报警阈值并保存 Flash v1

## 一、项目目标

本项目基于前一天完成的温湿度阈值报警系统，继续增加按键配置功能。

第24天的目标是：

- 通过按键修改温度报警上限
- 通过按键修改湿度报警下限
- 修改后的配置保存到 STM32 内部 Flash
- 复位或断电后配置仍然有效
- 报警判断使用 Flash 中保存的配置参数

相比第23天，本版本不再只使用固定阈值，而是让报警阈值具备可配置能力。

---

## 二、当前系统功能

当前系统已经具备：

- DHT11 温湿度采集
- OLED 实时显示
- 串口状态输出
- FreeRTOS 多任务调度
- 消息队列传递传感器数据
- 共享结构体保存当前系统状态
- Flash 保存配置参数
- IWDG 看门狗任务健康检测
- 温湿度报警状态判断
- 按键修改报警阈值

---

## 三、硬件连接

### DHT11

| DHT11 | STM32 |
|---|---|
| VCC | 3.3V |
| DATA | PB12 |
| GND | GND |

### OLED

| OLED | STM32 |
|---|---|
| SCL | PB6 |
| SDA | PB7 |
| VCC | 3.3V |
| GND | GND |

### 按键

按键采用上拉输入方式：

| 按键 | 功能 |
|---|---|
| KEY_MODE | 切换当前设置项 |
| KEY_UP | 当前设置项 +1 |
| KEY_DOWN | 当前设置项 -1 |

按键接法：

```text
GPIO 引脚 ---- 按键 ---- GND
