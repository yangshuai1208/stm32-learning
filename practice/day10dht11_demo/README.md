# STM32 DHT11 温湿度传感器读取（DWT 精确延时）

## 功能
- 使用单总线协议读取 DHT11 传感器的温湿度数据
- 使用 DWT 实现微秒级精确延时，满足时序要求
- 通过串口打印温湿度值（整数+小数）
- 包含数据校验，确保读取正确

## 硬件连接
- MCU：STM32F103C8T6（野火开发板）
- DHT11 数据引脚：PB12
- 串口：USART1（PA9 TX, PA10 RX）
- 系统时钟：72MHz

## 使用方法
1. 用 Keil 5 打开工程，编译下载
2. 打开串口助手（115200, 8N1）
3. 开发板每 2 秒读取一次 DHT11，并打印结果，例如：
   `湿度为 45.0 %RH，温度为 26.5 ℃`

## 代码结构
- `bsp_dht11.c/h`：DHT11 底层驱动（GPIO 配置、时序读写）
- `app_dht11.c/h`：应用层，读取数据并格式化输出
- `bsp_dwt.c/h`：DWT 精确延时（微秒/毫秒/秒）
- `bsp_led.c/h`：LED 控制（本例未使用，可扩展）
- `main.c`：主循环每 2 秒调用一次读取

## 注意事项
- DHT11 对时序要求严格，必须使用微秒级延时（DWT 或 SysTick）
- 读取间隔建议不小于 1 秒，否则传感器可能不响应
- 校验和错误时会打印“读取DHT11数据错误！”
