# BMP280 气压/温度传感器驱动（STM32，SPI 接口）

## 功能
- 通过 SPI 接口读取 BMP280 传感器的原始气压和温度 ADC 值
- 读取芯片内部的校准系数，并通过数据手册提供的公式进行温度与气压补偿计算
- 通过串口打印温度（℃）和气压（KPa）
- 支持过采样、滤波器、采样间隔等配置

## 硬件连接
- MCU：STM32F103C8T6（野火开发板）
- BMP280 模块（SPI 模式）：
  - VCC → 3.3V
  - GND → GND
  - SCK → PA5（SPI1_SCK）
  - SDA → PA7（SPI1_MOSI）
  - SDO → PA6（SPI1_MISO）
  - CS  → PA4（片选）
- 串口：USART1（PA9 TX, PA10 RX），115200
