# Day09 FreeRTOS任务调度与osDelay实验

## 一、实验目标
在 FreeRTOS 工程中继续完善基础任务，观察任务调度现象，并理解 `osDelay()` 在任务调度中的作用。

## 二、实验内容
今天完成了以下内容：

- 继续使用 LED 任务和串口任务
- 为任务增加运行计数变量
- 通过串口观察不同任务的执行次数
- 理解任务的运行态、就绪态和阻塞态
- 学习 `osDelay()` 对任务调度的影响

## 三、任务设计
### 1. LED任务
- 任务名：`ledTask`
- 功能：周期翻转 LED
- 周期：500ms

### 2. 串口任务
- 任务名：`uartTask`
- 功能：周期打印任务运行信息
- 周期：1000ms

## 四、知识点总结
通过今天的实验，主要学习了以下内容：

- FreeRTOS 任务调度的基本概念
- 任务状态的基本理解
- `osDelay()` 的作用
- `osDelay()` 与 `HAL_Delay()` 的区别
- 优先级对任务调度的影响

## 五、实验现象
下载程序后可以观察到：
- LED 任务按 500ms 周期闪烁
- 串口任务按 1000ms 周期打印日志
- 串口日志中可以看到两个任务计数值的变化

## 六、实验结论
今天完成了 FreeRTOS 基础任务调度现象的观察，理解了任务延时和任务切换的基本逻辑，为后续学习队列和任务通信打下了基础。
