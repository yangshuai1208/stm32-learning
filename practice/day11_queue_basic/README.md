# Day11 FreeRTOS队列通信实验

## 一、实验目标
在 FreeRTOS 工程中加入消息队列，实现 `sensorTask` 和 `uartTask` 之间的数据通信。

## 二、实验内容
今天完成了以下内容：

- 在 CubeMX 中创建消息队列
- `sensorTask` 周期产生模拟数据
- `sensorTask` 通过队列发送数据
- `uartTask` 阻塞等待队列消息
- `uartTask` 接收到数据后通过串口打印

## 三、任务设计
### 1. sensorTask
- 功能：生成数据并发送到队列
- 周期：500ms

### 2. uartTask
- 功能：从队列接收数据并打印
- 接收方式：阻塞等待

## 四、知识点总结
通过今天的实验，主要学习了以下内容：

- 队列的基本概念
- 队列在任务间通信中的作用
- 发送和接收任务的职责分离
- `osMessagePut()` 和 `osMessageGet()` 的基本使用方法

## 五、实验现象
程序运行后，LED 周期闪烁，串口助手可以周期性看到队列接收到的数据，说明任务间通信已经建立成功。

## 六、实验结论
今天完成了 FreeRTOS 从“任务拆分”到“任务通信”的进一步过渡，为后续学习更复杂的同步与通信机制打下了基础。
