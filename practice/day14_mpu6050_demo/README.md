# MPU6050 六轴传感器驱动（加速度+陀螺仪+温度，STM32，I2C，中断）

## 功能
- 通过 I2C 读取 MPU6050 的加速度（X/Y/Z）、陀螺仪（X/Y/Z）和温度数据
- 使用中断方式（数据准备好中断）通知 MCU 读取数据
- 通过串口打印原始数据和转换后的温度值（摄氏度）
- 支持 AD0 引脚控制从机地址（0x68/0x69）

## 硬件连接
- MCU：STM32F103C8T6（野火开发板）
- MPU6050 模块：
  - VCC → 3.3V
  - GND → GND
  - SCL → PB6（I2C2）
  - SDA → PB7（I2C2）
  - INT → PB8（中断引脚）
  - AD0 → PB9（可选，接地或高电平控制地址）
- 串口：USART1（PA9 TX, PA10 RX），115200
- 调试 LED：红色 LED 在中断触发时翻转（可选）

## 使用方法
1. 用 Keil 5 打开工程，编译下载
2. 打开串口助手（115200, 8N1）
