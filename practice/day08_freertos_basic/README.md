# Day08 FreeRTOS移植与基础任务创建

## 一、实验目标
在 STM32F103 工程中移植 FreeRTOS，并创建最基础的任务，验证任务调度功能是否正常。

## 二、实验内容
今天完成了以下内容：

- 在 CubeMX 中添加 FreeRTOS 中间件
- 创建 LED 任务
- 创建串口打印任务
- 使用 `osDelay()` 实现任务周期执行
- 验证 FreeRTOS 调度器能够正常运行

## 三、任务设计
### 1. LED任务
- 任务名：`ledTask`
- 功能：周期翻转 LED
- 周期：500ms

### 2. 串口任务
- 任务名：`uartTask`
- 功能：周期打印调试信息
- 周期：1000ms

## 四、知识点总结
通过今天的实验，主要学习了以下内容：

- FreeRTOS 的基本作用
- 任务的概念
- 调度器的基本工作方式
- `osDelay()` 的使用方法
- 裸机工程与 RTOS 工程的区别

## 五、实验现象
下载程序后，可以观察到：
- LED 按设定周期闪烁
- 串口每隔 1 秒输出一次调试信息

## 六、实验结论
今天完成了从裸机工程到 RTOS 工程的过渡，成功创建了最基础的任务，为后续学习队列、任务通信和多任务项目整合打下了基础。
