# Day22 Flash 配置参数保存 v1

## 一、项目目标

本项目基于前面已经完成的 STM32F103 + FreeRTOS 温湿度监测系统，新增内部 Flash 配置参数保存功能。

第22天的核心目标是：

- 定义系统配置结构体
- 使用 STM32 内部 Flash 保存配置参数
- 上电时自动读取 Flash 配置
- 如果 Flash 中没有有效配置，则加载默认配置并写入 Flash
- 让配置参数真正参与系统运行

本次实验重点不是保存温湿度历史数据，而是保存系统配置参数。

---

## 二、当前系统基础

当前项目已经具备：

- DHT11 温湿度采集
- OLED 实时显示
- 串口状态输出
- FreeRTOS 三任务结构
- 队列传递最新采样结果
- 共享结构体保存当前状态
- IWDG 看门狗与任务健康检测基础

第22天在此基础上新增 Flash 参数保存模块。
